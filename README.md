# 隐私政策

**更新日期：2026年3月24日**  
**生效日期：2026年3月24日**

烟台云都海鹰无人机应用技术有限公司（以下简称“我们”）作为“植保联盟飞手作业”应用程序（以下简称“本应用”）的运营者，深知个人信息对您的重要性，并会尽全力保护您的个人信息安全可靠。本隐私政策将向您说明我们如何收集、使用、存储、共享和保护您的个人信息，以及您如何行使您的权利。

请您在使用本应用前，仔细阅读并充分理解本政策。如您不同意本政策中的任何内容，请立即停止使用本应用。

## 一、我们收集的个人信息

在您使用本应用的过程中，我们会出于以下目的收集您的个人信息：

| 业务场景 | 个人信息类型 | 收集目的 |
|---------|------------|---------|
| 账号登录（如有） | 手机号、昵称 | 识别用户身份，提供飞手认证、作业记录同步功能 |
| 地图与作业定位 | 精确位置信息、经纬度 | 为您显示作业区域、规划航线、记录作业轨迹 |
| 设备安全保障 | 设备标识符（IMEI/Android ID/OAID）、设备型号、操作系统版本 | 保障应用安全运行，防止恶意攻击，进行设备兼容性适配 |
| 应用分析与优化 | 应用使用记录、点击事件、页面访问时长 | 分析用户行为，优化功能体验，提升稳定性 |
| 问题诊断 | 崩溃日志、错误日志 | 排查应用故障，修复问题，提高应用稳定性 |

**特别说明**：  
- 位置信息仅在您使用作业相关功能时（如查看地图、创建作业任务）才会收集，您可以随时通过系统权限管理关闭位置权限。  
- 设备标识符在您同意后用于安全风控，不会用于其他商业目的。

## 三、数据存储与跨境传输

1. **存储地点**：我们收集的个人信息将存储在中华人民共和国境内的服务器上。  
2. **存储期限**：我们仅在为实现本政策所述目的所必需的时间内保留您的个人信息，除非法律法规另有要求。  
3. **跨境传输**：我们不会将您的个人信息传输至境外。如未来因业务需要发生跨境传输，我们会单独征得您的授权同意，并履行相关法律程序。

## 四、我们如何保护您的信息

我们使用符合业界标准的安全防护措施（如SSL加密、访问控制、数据脱敏等）保护您的个人信息，防止数据遭到未经授权的访问、公开披露、使用、修改、损坏或丢失。

## 五、您的权利

您可以通过以下方式管理您的个人信息：

1. **查询/更正/补充**：您可以在应用内“个人中心”查看或修改您的手机号、昵称等信息。  
2. **删除**：您可以删除作业记录、收藏地点等非必要信息。若需删除账号，请联系我们进行注销。  
3. **撤回同意**：您可以通过手机系统设置关闭位置、存储等权限，撤回对相关个人信息收集的同意。  
4. **注销账号**：如需注销账号，请发送邮件至 `yunduhaiying@163.com` 或致电 `[联系电话]`，我们将在15个工作日内完成注销。  
5. **获取个人信息副本**：您可以通过上述联系方式联系我们，获取我们持有的您的个人信息副本。

## 六、未成年人保护

本应用主要面向从事农业植保作业的成年人。我们不会主动收集未成年人的个人信息。如您为未成年人，请在监护人指导下使用本应用。
